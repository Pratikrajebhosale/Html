# Simple HTML Web Page

##  Project Description

This project is a basic HTML web page created as part of an assignment. It demonstrates the use of fundamental HTML structure including headings, paragraphs, and proper document formatting.

##  Technologies Used

* HTML5

##  How to Run the Project

1. Download or clone the project files.
2. Locate the file named `index.html`.
3. Double-click the file OR open it using any web browser (Chrome, Edge, Firefox).
4. The webpage will display in your browser.

##  Features

* Basic HTML structure
* Proper use of tags
* Simple and clean layout

##  Notes

* Ensure all HTML tags are properly opened and closed.
* Tested in modern web browsers.
